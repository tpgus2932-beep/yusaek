# YUSAEK 관리자 대시보드 (Admin Dashboard)

**React**, **Vite**, 그리고 **Lucide Icons**를 사용하여 제작된 프리미엄 미니멀 블랙 & 화이트 관리자 대시보드입니다.

## ✨ 주요 기능

- **모던한 미니멀리즘**: 브랜드 아이덴티티를 살린 깔끔한 블랙 & 화이트 디자인과 프리미엄 타이포그래피(Outfit) 적용.
- **반응형 레이아웃**: 데스크탑, 태블릿, 모바일 등 모든 기기에서 최적화된 화면 제공.
- **다크 모드 지원**: 사용자 설정에 따라 라이트 모드와 다크 모드 간의 부드러운 전환 지원.
- **CSS 모듈화 (Modular CSS)**: CSS Modules를 사용하여 스타일 충돌을 방지하고 유지보수성 극대화.
- **컴포넌트 기반 아키텍처**: 기능별로 구조화된 폴더 구성으로 팀 협업 및 확장성 용이.
- **인터랙티브 대시보드**: 수익 추적, 최근 거래 내역, 활동 로그 등 관리자에게 필요한 핵심 기능 구현.

## 🛠 기술 스택

- **프레임워크**: React
- **빌드 도구**: Vite
- **스타일링**: CSS Modules (Vanilla CSS)
- **아이콘**: Lucide-React
- **타이포그래피**: Google Fonts (Outfit)

## 📁 프로젝트 구조

